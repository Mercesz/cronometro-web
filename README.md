# Cronômetro Web

Cronômetro responsivo feito com HTML, CSS e JavaScript. Possui botões de iniciar, pausar e zerar, com layout adaptado para mobile.

## Tecnologias
- HTML
- CSS
- JavaScript

## Como testar
Abra o `index.html` em qualquer navegador ou acesse o deploy online.
